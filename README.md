## 📖 核心理念 (Core Philosophy)

本專案旨在收錄那些能引導信徒反思、產生真實變革的證道資源。

### 1. 文字的消化效率與傳播廣度 (Textual Efficiency & Scalability)
目前教會多以影音紀錄證道，雖便於製作，但文字具備影音無法替代的優勢：
* **自主性與深度**：影音是線性的，聽眾必須跟隨講員的速度；而文字允許讀者**隨時停頓、回溯、反覆咀嚼**。
* **高效率檢索 (Searchable)**：文字內容具備可搜索性，信徒能透過關鍵字定位真理教導。
* **無遠弗屆的傳播 (Spreadable)**：文字檔案體積小、易於複製與跨平台分享。透過搜尋引擎與社群轉發，能更有效地**向外擴散**。

### 2. 復興的先決條件：反省與拆毀
基督徒皆知神的美好，但生命的轉變始於**反省 (Self-examination)**。
* **摩西的警誡**：效法摩西在離世前的切切叮嚀（申命記 31章），直面人性的軟弱與悖逆。
* **希西家的改革**：如同希西家王打碎銅蛇，屬靈復興必先經歷自省與拆毀偶像，而非廉價的歌頌。

### 3. 聖靈與真理的同在
**「讀經固然重要，但若沒有神的靈同在，真理便無法進入人心。」**
讀經是根基，聖靈是動力，而反省則是聖靈在心中結出的果子。

### 4. 信仰的實質：先有真理，後有神蹟
當前教會面臨一種危機，即信徒與教會群體逐漸失去改變社會的「鹽與光」特質。

* **次序的偏離**：若無法在生命中活出真理，往往會試圖透過提供「免費餐點」或社交活動來吸引人。這種模式雖能帶來人群，卻難以傳遞真理。
* **基督的典範**：**耶穌總是先傳講天國的道（真理），隨後才憐憫群眾、餵飽飢餓的人（神蹟/物資）**。若將次序顛倒，福音將淪為一種廉價的福利，而非靈魂的重整。
* **回歸本質**：筷子該拿的時候還是得拿，但重心該從「以物質吸引人」的慣性，轉而追求以真理轉化人。

3. **屬靈實質**：著重於生命的改變與真理的實踐，拒絕以物質利益掩蓋靈性貧乏。
