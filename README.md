## 📖 核心理念 (Core Philosophy)

本專案旨在收錄那些能引導信徒反思、產生真實變革的證道資源。
### 1. 文字的消化效率 (Textual Efficiency vs. Sequential Media)
目前教會多以影音（Video/Audio）紀錄證道，雖便於製作，但對接收者而言：
* **影音是線性的 (Sequential)**：聽眾必須跟隨講員的速度，難以即時停頓思索。
* **文字具備自主性**：文字允許讀者**隨時停頓、回溯、反覆咀嚼**。

### 2. 復興的先決條件：反省與拆毀
基督徒皆知神的美好，但生命的轉變始於**反省 (Self-examination)**。
* **摩西的警誡**：效法摩西在離世前的切切叮嚀（申命記 31章），直面人性的軟弱與悖逆。
* **希西家的改革**：如同希西家王打碎銅蛇，屬靈復興必先經歷自省與拆毀偶像，而非廉價的歌頌。

### 3. 聖靈與真理的同在
**「讀經固然重要，但若沒有神的靈同在，真理便無法進入人心。」**
讀經是根基，聖靈是動力，而反省則是聖靈在心中結出的果子。

### 4. 信仰的實質：鹽與光的生命，而非廉價的社交吸引
當前教會面臨一種危機，即信徒與教會群體逐漸失去改變社會的「鹽與光」特質。
* **現象的反思**：基督徒若無法在生命中活出真理的影響力，往往會退而求其次，試圖透過提供「免費餐點」或社交活動來吸引福音朋友。
* **導向的偏差**：這種模式雖能帶來人群，卻難以傳遞真理。若缺乏生命的震撼力，福音將淪為一種廉價的福利，而非靈魂的重整與救贖。
* **回歸本質**：筷子該拿的時候還是得拿， 但重心該從「以物質吸引人」的慣性，轉而追求以真理轉化人。

## 📂 收錄準則
1. **反省**：挑戰宗教慣性，不提供廉價安慰，只追求拆毀後的真實重建。拒絕盲目樂觀。
2. **聖靈同在**：超越枯燥教義，強調生命與神的直接連結。
3. **屬靈實質**：著重於生命的改變與真理的實踐，拒絕以物質利益掩蓋靈性貧乏的策略。
